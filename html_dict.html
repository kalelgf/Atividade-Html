Dicionário de Tags HTML

<!DOCTYPE>
Define o tipo de documento (HTML5).
<html>
Elemento raiz de um documento HTML.
<head>
Contém metadados como título, charset, links.
<meta>
Define metadados como codificação de caracteres.
<title>
Define o título da aba do navegador.
<link>
Usado para conectar arquivos externos, como CSS ou ícone.
<body>
Contém todo o conteúdo visível da página.
<header>
Define o cabeçalho da página ou seção.
<nav>
Define um menu de navegação.
<main>
Contém o conteúdo principal da página.
<section>
Define uma seção genérica de conteúdo.
<aside>
Define conteúdo complementar (como uma barra lateral).
<footer>
Rodapé da página ou de uma seção.
<h1> a <h6>
Títulos de diferentes níveis, do mais importante ao menos.
<p>
Define um parágrafo de texto.
<a>
Cria um hiperlink para outra página ou seção.
<img>
Insere uma imagem na página.
<ul> / <ol> / <li>
Listas não ordenadas, ordenadas e seus itens.
<form>
Define um formulário interativo para entrada de dados.
<input> / <textarea>
Elementos de formulário para entrada de texto e áreas maiores.
